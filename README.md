# 精品课实战

基于 Next.js 的精品课电商 wap 网站

## 开发

### 开发环境准备

- Node > 10
- global yarn

```BASH
# 首次安装
yarn install
# 开发启动
yarn dev
```

浏览器打开 [http://loalhost:3000](http://loalhost:3000)

## 技术栈

- Next.js 12.2.5
- React 18
